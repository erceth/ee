jsFlags

http://jsflags.ericethington.com

Robot AI tanks and game engine written 100% in nodeJS and javascript from scratch. A rewrite of this BTZFlag Python game. Tanks move about a 2d world shooting enemy tanks and stealing flags for points. Actions of tanks are recieved from a Javascript written artificial intelligence. Goal was to write the game with as few dependencies as possible to make the game super portable and easy to set up. This project is written to be a fun hackathon where each member creates an AI and battle each other. I also plan to make several improvments from the original python game including a manual mode. Still in development. View in Chrome. Check out the source.

About Me

Hello, my name is Eric Ethington. Welcome to my website. This site serves as the front page of ericethington.com. The purpose of ericethington.com is two parts. One, a place for me to show off my personal projects and two, to help people get to know me a little better. From now on every time I create a new personal project I will put a link here for people to refer to.

I’m a web developer. I am graduated in Computer Science from Brigham Young University in April 2013. I have been working as a front end web developer ever since. While in college I had an internship for a year where I got my taste for web development and decided that’s that I wanted as a career.

I also have a strong interest in Psychology. I graduated with a minor simply because I liked it. I read psychology type book often and I am fascinated with how people work.

I have a smart and beautiful wife that I love very much. We are best friends. We love spending time with each other and sharing our big ideas with each other.

That’s a little bit about me. I hope you find some interest in me an my projects.
